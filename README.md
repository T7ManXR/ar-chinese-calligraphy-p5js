# AR Chinese Calligraphy with P5.JS & ML5.JS
AR (Augmented Reality) is an increasingly ubiquitous and popular technical method of
representation where digital elements overlay the real environment. In this artist statement, I will
outline the inspiration, process, and demonstrations behind an innovative calligraphy tool.
## Inspiration and References
Firstly, I have been writing Chinese calligraphy since childhood and produced Fai Chun/Chunlian on my
own. These are red paper banners that stick around the main door or entrance of buildings to wish
for good fortune and prosperity as a Chinese New Year tradition. If brush ink calligraphy can be
replicated with AR, then anyone with the source code can draw without purchasing physical
writing tools. Using Hooke’s Law to simulate brush strokes, a creative coding blog, is another
inspiration source by integrating physics to elevate this experience (Moussa, 2021)[^1].
### Controls
Instead of the mouse cursor, pinch detection from either hand controls the brush strokes, adjusting brush size
and on/off toggle. Below are two sample demonstrations with handpose and AR.

- Draw with black Ink using pinch gestures (‘v‘ to toggle between black/red canvas, c‘ to draw, ‘x‘ to stop,
‘z‘ to reset) 
- Use number keys (1-9) to change the brush stroke thickness

[^1]: [Simulating Brush Strokes with Hooke's Law in P5JS and Processing](https://www.gorillasun.de/blog/simulating-brush-strokes-with-hookes-law-in-p5js-and-processing/)
